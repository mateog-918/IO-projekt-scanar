# Temat projektu
Proszę wyobrazić sobie, że jesteśmy w firmie programistycznej, gdzie otrzymujemy następujące zamówienie:

Państwa klientem jest firma przemysłowa zarządzająca dużą fabryką. Celem wdrożenia systemu jest skuteczna kontrola wejść na teren zakładu pracy. Aktualnie dostęp dla pracowników odbywa się przy użyciu kart magnetycznych, jednak coraz częściej obserwowane są nadużycia polegające na przekazywaniu kart dostępowych między pracownikami. Powoduje to błędne naliczanie czasu obecności oraz fałszywe raporty dotyczące czasu pracy.

Firma oczekuje rozwiązania, które pozwoli ograniczyć te nadużycia poprzez integrację dwóch mechanizmów: skanowania ważnej przepustki (QRcode) oraz automatycznej identyfikacji osoby na podstawie wizerunku twarzy w momencie wejścia na teren fabryki. Oprogramowanie powinno rejestrować wszystkie próby wejścia, odnotowywać szczegóły związane z identyfikacją i zapewniać raporty dotyczące efektywności oraz wykrytych nadużyć. System powinien także obsługiwać rejestr uprawnień pracowników oraz listę osób uprawnionych do wejścia.
